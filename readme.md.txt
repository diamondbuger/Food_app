# PizzaHub - MERN Food Delivery App

A full-stack food delivery application built with MERN (MongoDB, Express, React, Node.js).

## Features

- ✅ User Authentication with JWT
- ✅ Menu Management (Admin)
- ✅ Shopping Cart
- ✅ Order Management
- ✅ Inventory Management
- ✅ Cash on Delivery Payment
- ✅ Admin Dashboard
- ✅ Strong Password Validation
- ✅ Cookie-based Authentication

## Tech Stack

### Frontend
- React.js
- React Router
- Axios
- Context API

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- bcryptjs for password hashing

## Installation

### Prerequisites
- Node.js (v14 or higher)
- MongoDB
- npm or yarn

### Frontend Setup

```bash
cd frontend
npm install
npm start
